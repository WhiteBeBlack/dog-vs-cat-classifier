# Dog vs Cat Classifier 🐶🐱

This project is a machine learning model that classifies images as dogs or cats.

## Files
- DogVsCatClassifier.ipynb – Jupyter notebook containing the model
- cat.jpg – sample cat image
- dog.jpg – sample dog image

## Tools Used
- Python
- TensorFlow / Keras
- Jupyter Notebook

## Goal
Train a CNN model to classify dog and cat images.
